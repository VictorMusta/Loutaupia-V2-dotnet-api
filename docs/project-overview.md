# Project Overview: Loutaupia-V2-dotnet-api

## Executive Summary
**Loutaupia-V2-dotnet-api** est un service backend robuste développé avec **.NET 8.0**. Il sert d'API principale pour l'écosystème Lootopia, gérant la logique métier, les données et les interactions avec les clients via une interface RESTful. Le projet est structuré de manière moderne, utilisant des approches par fonctionnalités (Features) et une architecture propre.
